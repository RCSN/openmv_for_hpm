# DMA Manager

## 概述

DMA Manager 展示了如下功能。
- 申请/释放DMA资源
- 使用相应DMA资源的中断
- 安装DMA资源的中断回调函数

## DMA Manager使用提示

- DMA Manager组件内部已经内置了中断处理函数，用户程序中不应当再声明中断处理函数

## 硬件设置

无特殊设置
