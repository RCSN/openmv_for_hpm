# eMMC 底层 API 示例

## 概览

本示例提供如下功能:
- 写最后一个扇区并读回来校验
- 写最后1024个扇区并读回来校验

## 板级设置

本示例基于eMMC转TF卡

## 运行示例

当示例成功运行后，会显示如下菜单，请输入1、2进行测试。

```console

-----------------------------------------------------------------------------------
*                                                                                 *
*                   eMMC Low-level test demo                                      *
*                                                                                 *
*        1. Write & Read the last block                                           *
*        2. Write & Read the last 1024 blocks                                     *
*                                                                                 *
*---------------------------------------------------------------------------------*




```
